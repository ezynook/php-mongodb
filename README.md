# Deploy
Pull project
```bash
git clone https://github.com/ezynook/php-mongodb.git
```
Change path to project
```bash
cd php-mongodb
```
Start and build Container
```bash
docker-compose up -d --build
```
Access on Browser
```bash
http://localhost
```
